Class: FilletedPathModel

FilletedPathModel(pathPoints, filletRadii, closed)

Creates a model from a series of points, filleting corners with the supplied radii

Constructor

new FilletedPathModel(pathPoints, filletRadii, closed)

Create the model
Parameters:
Name Type Default Description
pathPoints Array The series of absolute points that represent the corners of the shape
filletRadii Array The radii to use for each corner (starting at 0 for the first 2 points in PathPoints), any unsupplied points will be 0 (no fillet)
closed Boolean false [closed=false] - Close the model using the first and last points
